📖 О плагине

Feed2Mail — это PHP-плагин для создания и управления email-рассылками на основе RSS-лент. Позволяет автоматически генерировать красивые HTML-письма из новостей и отправлять подписчикам.


Версия: 1.1

Дата релиза: 14 апреля 2026

Разработчик: TCSE студия

Сайт: tcse-cms.com

⚙️ Системные требования

  • PHP ≥ 7.4 (рекомендуется 8.0+)
  • Расширения: json, session, fileinfo, mbstring
  • SimpleXML — для парсинга RSS
  • PHPMailer (опционально, для SMTP)
  • Права на запись в папку /data и /assets/uploads

🚀 Быстрая установка

  1. Скопировать папку feed2mail в /plugins/tcse/
  2. Убедиться, что папки data/ и assets/uploads/ доступны для записи
  3. Перейти в админку: /plugins/tcse/feed2mail/admin/
  4. Ввести пароль по умолчанию: MySecretPass123
  5. Настроить SMTP или оставить php_mail
  6. Вставить код виджета на сайт

     data-site-url="https://ваш-сайт.ru" 
     data-plugin-path="/plugins/tcse/feed2mail"></div>
<script src="https://ваш-сайт.ru/plugins/tcse/feed2mail/assets/embed.js"></script>

Особенности:

  • ✅ Автоопределение пути к API
  • ✅ Защита от ботов (rate limiting + honeypot)
  • ✅ Двойной opt‑in (подтверждение по email)
  • ✅ CORS — работает на любых доменах

📡 RSS → Email

Поддерживаемые форматы: RSS 2.0, Atom

Настройки генерации:

  • Обрезка описания до N символов

Режимы отправки:

  • php_mail — стандартная PHP mail()

Особенности:

  • Подстановка логотипа
  • Ссылка на отписку в каждом письме
  • Статистика отправки (успешно/ошибки)

🎨 Настройка шапки и подвала

В файле data/template_settings.json или через админку → Настройки писем:

{
  "site_name_in_email": "Блог Талика",
  "header": "<div style=\"background: #1a1a2e; padding: 20px;\">
            <h1>{{SITE_NAME}}</h1>
            </div>",
  "footer": "<div>© {{YEAR}} {{SITE_NAME}}. 
            <a href=\"{{UNSUBSCRIBE_LINK}}\">Отписаться</a></div>",
  "logo_url": "https://site.ru/logo.png",
  "logo_position": "top",
  "use_header": true,
  "use_footer": true
}

Доступные переменные: {{SITE_NAME}}, {{SITE_URL}}, {{YEAR}}, {{LOGO_URL}}, {{UNSUBSCRIBE_LINK}}

🔌 API Endpoints

POST /api/subscribe.php — новая подписка

{
  "email": "user@example.com"
}

GET /api/subscribe.php — подтверждение подписки

?token=xxx&email=user@example.com

GET /api/unsubscribe.php — отписка

?token=xxx&email=user@example.com

🐛 Устранение неполадок

  • Кракозябры в письмах → Проверить кодировку файлов (UTF-8 без BOM)
  • Не приходит письмо → Проверить SMTP или переключить на php_mail
  • Ошибка CORS → Проверить заголовки в api/subscribe.php
  • Не загружается логотип → Проверить права на assets/uploads/ (755)
  • Не работает виджет → Открыть консоль браузера (F12), смотреть ошибки
